Este foi um projeto desenvolvido durante o 4º Semestre durante a disciplina Programação De Sítios para Internet.

A ideia foi a criação de um site de um hotel aplicando técnicas de HTML, CSS e JavaScript desenvolvidos durante a sala de aula. O site exibe uma interface com elementos que compõem um hotel e todos seus serviços separadas por abas individuas, facilitando assim, a boa compreensão do usuário do site.

Foram utilizadas uma interfaces limpas com paletas de cores claras que visavam combinar com os elementos do hotel. Visto isso, as fontes e imagens aplicadas no site são exibidas de forma objetiva demonstrando um site simples e funcional.

No projeto de desenvolvimento do site, elaboramos um hotel fictício chamando Campos Elísios, onde exibe um grande hotel de luxo com uma estrutura bastante extravagante. O site do hotel mostra o seus serviços oferecidos, com grandes salas onde podem ser oferecidas festas e atender a todos os públicos, Em sua versão mobile, as letras ficam em um bom tamanho de fonte, as seções ficam bem divididas, tudo para uma boa leitura e entendimento.

O código foi desenvolvido semanticamente, visando atender a todas a exigências técnicas de boas práticas de programação. Desta forma, podendo fazer manutenção com facilidade, incluir novos elementos e tornar o site o mais objetivo possível.
